# SWM Desk - C++ Qt6 Streaming Dashboard

Cross-platform C++ Qt6 control panel for OBS overlays and streaming animations.

## Structure

- **app**: Bootstrapping and main execution entry points
- **presentation**: QML user interfaces and view models
- **core**: Core graphics and timeline logic
- **infrastructure**: Cross-platform database and websocket implementations
- **integrations**: OBS Studio plugin and Stream Deck configurations
- **marketplace**: Theme packages and template updates
- **storage**: Persistent assets and logs
