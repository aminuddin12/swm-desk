# Naming Conventions

Directory and file naming rules:

- **Classes / Structs**: PascalCase (e.g. `AppBootstrap`, `DatabaseManager`).
- **Variables / Fields**: camelCase (e.g. `clientSockets`, `activePresetId`).
- **Functions / Methods**: camelCase (e.g. `selectPreset()`, `initDb()`).
- **Files**: kebab-case or matching the main class (e.g. `main.cpp`, `AppBootstrap.cpp`).
- **Interfaces**: Prepended with capital `I` (e.g. `ILogger.h`, `IDatabase.h`).
