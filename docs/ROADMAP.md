# SWM Desk Roadmap

Outline of SWM Desk platform milestones:

- **Milestone 1**: Dynamic Service Registry and App Bootstrapping.
- **Milestone 2**: Core Rendering Engine and Animation Controller.
- **Milestone 3**: QML Sidebar UI and dynamic feature modules (Dashboard, Templates, Assets).
- **Milestone 4**: Plugin Loader SDK and OBS/Stream Deck bridge.
- **Milestone 5**: Marketplace Integration and Package Installer.
- **Milestone 6**: Licensing, monetization, and public SDK release.
